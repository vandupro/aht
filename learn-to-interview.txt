 1. Cấu trúc thư mục trong magento 2
  + bin: Chạy các lệnh CLI
  + generated: magento sinh code khi chay lenh setup:di:compile
  + lib: thu muc chua cac thu vien
  + pub: thu muc public, cho phep truy cap cac file tu ben ngoai, chua ma html, css, js, index.php
  + var: chua cac cai tam thoi khi ung dung chay: cache, log, view_preprocessed( chua cac file less, phtml, duoc sinh ra khi chay lenh setup:static-content:delog -f
